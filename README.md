# SMSGateway
A rpi powered sms gateway with api to send a sms from any device like esp8266 or smartphone.
It comes with a easy to use Webinterface for configuring stuff.




# PARTS
* RPI Zero
* Sim800L GSM Module
* Activated SimCard
* USB2Serial (TTL)
* Wires
* PNP Transistor
* 3 leds
* 4 resistors
